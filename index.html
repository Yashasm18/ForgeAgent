Forge protocol

Memory is a promotion, not a cache.

A candidate cannot become a reusable skill because it looks plausible. It needs policy clearance, isolated proof, and a durable decision record.

01 / GAPPlan the task

Identify the missing capability and its dependencies.

02 / PROPOSEConstrain creation

Generate a small tool plus explicit proof cases.

03 / GATEReject unsafe code

Block filesystem, network, dynamic execution, and unsafe imports.

04 / PROVEIsolate evidence

Run every deterministic case in a fresh timeout-bound process.

05 / REUSEPromote trust

Version, lineage, proof, rollback, and reuse become memory.
